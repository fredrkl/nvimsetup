# My Neovim setup

## Requirements

- [Nerdfonts](https://www.nerdfonts.com/font-downloads)

## Setup

- Install neovim
- Install Nerdfonts
- Run `:checkhealth lazy` for `lazy vim` plugin verification
- Install [Packer](https://github.com/wbthomason/packer.nvim?tab=readme-ov-file#quickstart)
